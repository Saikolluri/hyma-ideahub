# Hyma Ideahub

Ideahub is a core service of **Hyma Technologies** designed to collect, validate,
and manage startup ideas from founders in Tier-2 and Tier-3 cities.

## Purpose
- Collect startup ideas from aspiring founders
- Enable structured idea submission
- Act as the foundation for future legal, advisory, and product services

## Architecture (Current)
- Dockerized application
- CI pipeline using GitHub Actions
- Images published to Docker Hub
- Cloud-agnostic deployment ready

## DevOps Highlights
- Automated Docker builds on every push
- Secure Docker Hub authentication
- Production-ready container image

## Status
- CI completed
- Local testing successful
- Cloud deployment ready when required
